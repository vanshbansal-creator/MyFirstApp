# MyFirstApp 🚀

## 📱 About

MyFirstApp is a beginner Android project built using Kotlin.
It counts button clicks and navigates to a second screen after reaching a limit.

---

## ✨ Features

* Button click counter
* Limit-based navigation to second activity
* Simple and clean UI
* Activity lifecycle handling

---

## 🛠 Tech Used

* Kotlin
* Android Studio
* XML (UI Design)

---

## 🎯 What I Learned

* Basics of Android development
* Handling button click events
* Using Intents to switch activities
* Managing app flow
